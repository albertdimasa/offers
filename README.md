# Offers — Syirkah Aqiqah Dan Qurban Surabaya

Landing page for an **Aqiqah & Qurban** catering service in Surabaya, Gresik, and Sidoarjo, East Java.

## Tech Stack

- Vanilla HTML, CSS (Tailwind CSS via CDN), JavaScript
- Deployed via GitHub Actions

## Directory Structure

```
├── aqiqah/
│   ├── index.html        # Main landing page
│   ├── aqiqah.jpg        # Hero image
│   └── favicon.*         # Favicon assets
└── .github/workflows/
    └── deploy.yml        # CI/CD deployment
```

## Local Development

Just open `aqiqah/index.html` in a browser. No build tools required.

## Deployment

Push to `main` branch to trigger automatic deployment via GitHub Actions.
