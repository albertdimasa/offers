# Offers — Landing Page Demo Collection

Collection of standalone landing page demos (offer sites) for different local businesses in East Java.

## Demos

| Directory | Business | Service |
|---|---|---|
| `aqiqah/` | Syirkah Aqiqah Dan Qurban Surabaya | Aqiqah & Qurban catering |
| `zottex/` | ZOTTEX | Custom apparel & sablon kaos |
| `index.html` (root) | CV. Vivi Aris | AC service & maintenance |

Add a new demo by creating a new directory with an `index.html`.

## Tech Stack

- Vanilla HTML, CSS (Tailwind CSS via CDN), JavaScript
- FontAwesome 6.4 (CDN) — Google Fonts where needed
- Deployed via GitHub Actions (SSH port 2341)

## Local Development

Open any `index.html` in a browser. No build tools or server required.

## Deployment

Push to `main` branch to trigger automatic deployment to VPS.
