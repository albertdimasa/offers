Informasi teknis seputar pengerjaan dan garansi ZOTTEX

Berapa minimal jumlah pemesanan kaos di ZOTTEX?

Minimal order standar adalah 12 Pcs untuk 1 desain. Untuk pembuatan sampel/prototype produk sebelum pemesanan masal, Anda bisa menghubungi admin teknis via WhatsApp.

Bagaimana jika file desain yang saya miliki pecah atau kurang jelas?

Tim desainer ZOTTEX memberikan layanan bantu *redesign* / tracing file gambar menjadi format vector HD secara **GRATIS** untuk setiap pemesanan di atas 24 Pcs.

Apakah ada garansi garansi hasil cetak?

Ya. Kami memberikan Garansi 100% Ganti Baru jika terdapat cacat jahitan, sablon terkelupas, atau ketidaksesuaian jumlah spesifikasi pesanan.
